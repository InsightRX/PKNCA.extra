# Get parameter names from a run_nca() output object

Get parameter names from a run_nca() output object

## Usage

``` r
get_parameter_names_from_nca_data(nca_data)
```

## Arguments

- nca_data:

  output data from NCA, generated using
  [`run_nca()`](https://insightrx.github.io/PKNCA.extra/reference/run_nca.md)
